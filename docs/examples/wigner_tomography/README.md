# Worked example — Wigner tomography of a displaced (coherent) state

A complete, end-to-end **digital-twin** demonstration of motional-state Wigner tomography
for a coherent state of *fixed amplitude and phase*, via the characteristic-function /
Ramsey route of [`docs/notes/strobo_grating_transfer_function.md`](../../notes/strobo_grating_transfer_function.md)
(§6–7). Generated by [`spike/twin_wigner_tomography.py`](../../../spike/twin_wigner_tomography.py).

> ⚠️ **Provenance.** The raw data here is **SIMULATED by the digital twin — it is NOT a real
> measurement.** It lives under `docs/examples/` (deliberately separate from `sources/data/`,
> the real DAQ path), and every file header repeats this. Regenerate with
> `python -m spike.twin_wigner_tomography`.

## The experiment (simulated)

| quantity | value |
|---|---|
| prepared state | coherent $\lvert\gamma\rangle$, $\gamma = 1.3\,e^{i\pi/4} = 0.919+0.919i$ |
| readout | spin-dependent-force / Ramsey characteristic-function readout, two quadratures |
| contrast | $C = 0.90$ |
| shots | $M = 500$ per probe point per quadrature |
| probe grid | $\beta$ on the disk $\lvert\beta\rvert\le4$, uniform spacing $h\approx0.33$ (441 points) |
| reconstruction grid | $\alpha\in[-2.5,2.5]^2$, $51\times51$ |

Note on reach: a single two-pulse Ramsey samples only $\lvert\Delta\beta\rvert\le2\eta\approx0.78$
— too small to cover the characteristic-function support. This example therefore uses the
**spin-dependent-force / extended-reach route** (a calibrated conditional displacement that
scans $\beta$ over a controllable region, up to $\sim N\eta$), as the tomography route in the
note requires.

## The four analysis steps

1. **Prepare** the known state $\lvert\gamma\rangle$.
2. **Measure** $\chi(\beta)=\mathrm{Tr}[\rho D(\beta)]$ on the probe grid. For each $\beta$,
   reading the qubit in two bases gives populations
   $P_x=\tfrac12(1+C\,\mathrm{Re}\,\chi)$ and $P_y=\tfrac12(1+C\,\mathrm{Im}\,\chi)$; each is
   sampled with **shot noise** ($M$ repetitions → binomial counts $n_x,n_y$). → raw counts.
3. **Reconstruct** the characteristic function from the counts,
   $\hat\chi(\beta)=\big[(2n_x/M-1)+i(2n_y/M-1)\big]/C$, then the **Wigner function** by its
   2-D Fourier transform, $W(\alpha)=\frac{1}{\pi^2}\!\int d^2\beta\,\hat\chi(\beta)\,e^{\alpha\beta^*-\alpha^*\beta}$.
4. **Validate**: the displacement is the centroid $\langle\alpha\rangle=\gamma$. Recovered
   here: $\lvert\gamma\rvert=1.30$ (input $1.30$), $\arg\gamma=0.79$ rad (input $0.785$);
   peak Wigner error $\approx 0.035$ (set by the $M=500$ shot noise).
